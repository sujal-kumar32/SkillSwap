 # SkillSwap – Peer-to-Peer Skill Exchange Platform

SkillSwap is a full-stack web application designed to connect individuals who want to **learn, teach, and exchange skills** in a collaborative environment. Instead of traditional paid learning platforms, SkillSwap enables users to **trade knowledge directly with others**, making learning more accessible and community-driven.

# Key Features

*  User Authentication (Signup/Login system)
*  User Profiles with skills and interests
*  Skill Exchange System (learn ↔ teach)
*  Post creation, likes, and interactions
*  Messaging & social interaction features
*  Notifications system
*  Skill credibility & growth tracking (planned advanced feature)

# Tech Stack

* **Frontend:** React.js, HTML, CSS, Bootstrap
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Atlas)
* **Other Tools:** REST APIs, Git, Postman

# Project Goal

The goal of SkillSwap is to build a **community-driven learning ecosystem** where users can grow their skills without financial barriers, while also showcasing their expertise and building meaningful connections.
